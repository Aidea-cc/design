# Design 设计文档仓库

> 用于存放UI/UX设计文档、设计规范、原型设计等内容

## 📁 目录结构

```
design/
├── docs/           # 设计文档
├── guidelines/     # 设计规范
├── prototypes/     # 原型设计
├── assets/         # 设计资源
└── templates/      # 设计模板
```

## 📝 文档分类

### 设计文档 (docs/)
- 产品设计文档
- 交互设计文档
- 视觉设计文档

### 设计规范 (guidelines/)
- 设计系统规范
- 组件设计规范
- 品牌设计规范
- 响应式设计规范

### 原型设计 (prototypes/)
- 高保真原型
- 交互原型
- 用户流程图

### 设计资源 (assets/)
- 图标库
- 字体资源
- 配色方案
- 设计素材

### 设计模板 (templates/)
- UI组件模板
- 页面模板
- 文档模板

## 🎯 使用说明

1. 按照目录结构分类存放设计文档
2. 文档命名规范：`项目名_文档类型_版本号.扩展名`
3. 重要文档变更需在提交信息中说明

## 📌 注意事项

- 所有设计文档应保持最新版本
- 废弃文档请移动到 `archive` 目录
- 协作设计请遵循团队设计规范
